01

Review before sign-in

Check API Permissions and Changelog first, then use the Microsoft popup sign-in. First-time tenant access requires admin consent for the delegated Graph scopes.

02

Pick your object type

Choose User, Group, Device, App, or CA Policy in the left rail, then search by name, UPN, or policy title.

03

Load the graph

Click one search result to render the live relationship graph and its linked operational context.

04

Inspect and pivot

Click nodes for details, copy IDs, use Entra deep links, and double-click user/group nodes to pivot deeper.

05

Run group impact before delete

For groups, open the Impact view to review blockers, warnings, remediation guidance, owners to contact, and checklist progress saved per group.

06

Export and share

Export the current graph or the impact analysis as JSON/CSV and attach it to tickets, CAB notes, or governance reviews. Idle sessions warn before automatic sign-out.
